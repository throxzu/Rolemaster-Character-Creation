# Talents

Rolemaster Core Law talent data — extracted from Chapter 4.

---

## Mechanics

- Talents cost **Development Points** (same as skills), fixed cost for all professions.
- **Flaws** have negative DP cost — they give back points to spend elsewhere.
- May be taken at character creation or during level advancement.
- A flaw cannot be taken unless it actually hinders the character.
- Talents with a background/source requirement: starting talents should be in the character's background; later talents should arise from experiences.
- **Upgrading**: pay the difference between the new tier cost and the current tier cost.
- **Restricted talents** (only apply in certain situations): cost is halved.

### Table 4-0: Maximum DP per Single Talent

| Power Level | Max DP |
|-------------|--------|
| Average | 10 |
| Superior | 20 |
| Heroic | 30 |
| Legendary | 40 |
| Epic | 50 |

Default Power Level is **Superior** (max 20 DP per talent) unless GM decides otherwise.

### Tier Costs

Most talents have multiple tiers (I, II, III…). The cost listed is per tier (e.g., `7/Tier` means Tier I = 7, Tier II = 14, Tier III = 21). Some talents have flat costs or use a two-value format (e.g., `10/20` means Tier I = 10, Tier II = 20).
