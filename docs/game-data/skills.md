### Skill Bonus Formula

```
Total Skill Bonus = Rank Bonus + Knack + Professional Bonus + Stat Bonus + Special Modifiers
```

- **Stat Bonus** = category stat 1 + category stat 2 + individual skill stat (three stats total)
- **Battle/Combat/Magical Expertise** categories use no stats at all
- **Rank Bonus** — see Table 3-0b
- **Knack** — +5 fixed bonus (chosen from profession list; 2 knacks per character)
- **Professional Bonus** — +1/rank up to max +30 (in 10 skills chosen from profession list)
- **Untrained** — –25 penalty when a character has 0 ranks (eliminated by first rank)

### Table 3-0b: Skill Rank Bonus

| Ranks | Bonus per rank |
|-------|---------------|
| 0 | −25 (untrained) |
| 1–10 | +5/rank |
| 11–20 | +3/rank |
| 21–30 | +2/rank |
| 31+ | +1/rank |

Max from ranks 1–10: +50 | 11–20: +30 | 21–30: +20 | then +1 per rank beyond 30.

### Table 3-0c: Skill Similarity Penalties

Used when substituting one skill for another the character lacks.

| Skill relation | Penalty |
|----------------|---------|
| Same skill, different specialization | −25 |
| Same category, different skill | −50 |
| Different category | −75 |

| Similarity (applied in addition) | Penalty |
|----------------------------------|---------|
| Very similar (same techniques AND same focus) | 0 |
| Similar (same techniques OR same focus) | −25 |
| Slightly similar (little overlap) | −50 |

### Selected Skill Notes (mechanically significant)

**Body Development** [Brawn/Co] — Bonus added to race base hits for total hit points. Character dies at negative hits exceeding this bonus. Also used for Endurance rolls.

**Ambush** [Subterfuge/In] — Adjusts critical strike roll up or down by up to character's Ambush ranks. Usable in melee if foe is unaware (ranks halved). No separate maneuver roll required.

**Power Development** [Power Manipulation/Co] — Bonus determines total Power Points available for spellcasting.

**Spell Lists** [Spellcasting/Me] — Ranks = maximum spell level castable from that list. Ranks + realm stat bonus added to Spellcasting Roll. Full skill bonus used for Spell Mastery (modifying spells on the fly).

**Riding** [Animal/Pr] — Unskilled rider can do nothing except perception/conversation while mount moves. Sprint/faster requires riding maneuver every round to avoid being thrown. Other actions while riding use full pace penalty from Table 5-3.

**Intense Training** — May purchase 3+ ranks/level in one skill by paying 2nd-rank cost for all ranks beyond the first. GM may require access to teacher/facilities. Cannot exceed ranks = twice character level (except Lore).

**Complementary Skills** — Secondary skills add their ranks (halved for each additional after first) to the primary skill maneuver roll.

**Conflicting Skills** — Active character gets modifier = 100 − passive character's final result.
